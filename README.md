# FRC Programming Resources

*   [WPILib](https://docs.wpilib.org/en/stable/) - This is the library we use for robotics (worth reading the whole thing once).
*   [Controls Engineering for FRC](https://file.tavsys.net/control/controls-engineering-in-frc.pdf) - (I love this doc).
*   [ReCalc](https://www.reca.lc/) - Helps calculate starting parameters for feedback feedforward.
*   [Yet Another Generic Swerve Library](https://docs.yagsl.com/)
*   
### Path Planning & Autonomous
*   [Choreo](https://choreo.autos/) - Physics-based trajectory optimizer (great for Swerve).
*   [PathPlanner](https://pathplanner.dev/) - The standard GUI for creating autonomous paths.

### Vision
*   [PhotonVision](https://photonvision.org/) - Open-source computer vision for AprilTags and game pieces.
*   [Limelight Docs](https://docs.limelightvision.io/) - Documentation for Limelight cameras.

### Logging & Debugging
*   [AdvantageScope](https://advantagescope.org/) - The best tool for viewing log files and 3D robot replays.
*   [AdvantageKit](https://advantagekit.org/) - Framework for "Log-Replay" simulation.

### Hardware Libraries
*   [CTRE Phoenix 6](https://v6.docs.ctr-electronics.com/) - For TalonFX, Kraken, and CANcoders.
*   [REVLib](https://docs.revrobotics.com/revlib) - For SPARK MAX/Flex and NEO motors.

### Other
*   [Statbotics](https://statbotics.io/) - Data analysis API for match predictions and scouting.
